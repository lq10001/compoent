## 我有话说
1. 在开发产品的过程中，我认为比较快速的方式，应该是功能组件的重复利用，并加上自动化工具。
2. 经过很多项目洗礼的组件，是比较成熟的。新项目中能直接使用，必定会减少很多时间。
3. 平时开发产品的过程中，抽象了很多公共的组件。虽然通过git方式分享了出来，UI组件也提供预览图，但是其他同事也很难快速查询到所需组件。
